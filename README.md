# 猜“言”大作战

## 项目简介

“猜‘言’大作战”是一个面向大学本科生的方言游戏网站。项目以娱乐体验为主，通过方言词义选择、方言来源判断、普通话反向猜方言等玩法，让用户在轻松答题的过程中接触不同地区的方言表达和文化背景。

当前阶段以搭建可展示概念的 Demo 为目标，优先完成完整、可运行的游戏流程，再逐步扩展用户系统、排行榜、投稿、音频题等功能。

## 核心玩法

### 猜词义

展示一个方言词或方言短句，用户从四个普通话释义中选择正确答案。

### 猜地区

展示一个方言词或表达，用户判断其主要使用地区。

### 反向翻译

展示普通话词义，用户从多个方言表达中选择对应答案。

## 当前阶段目标

第一阶段需要完成以下基础闭环：

```text
进入首页
→ 选择玩法
→ 开始答题
→ 判断答案
→ 查看方言解释
→ 完成本局
→ 查看得分和结果
```

第一阶段重点是跑通核心体验，不追求一次性完成所有功能。

## 项目目录

```text
dialect-game/
├─ README.md
├─ docs/
├─ frontend/
├─ backend/
├─ content/
├─ assets/
└─ presentation/
```

各目录用途如下：

- `docs/`：项目方案、页面流程、会议纪要和测试记录
- `frontend/`：前端代码
- `backend/`：后端代码
- `content/`：题库、方言资料和内容来源
- `assets/`：图片、图标和 UI 参考图
- `presentation/`：展示材料和汇报文件
