# 猜“言”大作战 · 前端

“猜‘言’大作战”是一个方言文化互动游戏。本目录包含 v0.1 前端 Demo：用户可以从首页进入“方言词义猜猜看”，完成一道本地示例题，并查看答题反馈与结果。

## 技术栈

- React
- Vite
- JavaScript
- React Router
- 普通 CSS

## 本地运行

请先安装 Node.js，然后在本目录运行：

```bash
npm install
npm run dev
```

构建生产版本：

```bash
npm run build
```

## v0.1 已实现功能

- 首页、玩法说明、答题、结果、关于和 404 页面
- “方言词义猜猜看”玩法入口
- 本地 JSON 示例题读取
- 选择、提交、判断、弹窗解释和结果展示流程
- 未选择答案提示、提交后锁定答案和结果页空状态保护
- 基础响应式布局与键盘焦点样式
